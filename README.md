<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如果在 Docker 容器中运行，推荐的基础镜像是 Alpine (3.15)</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">硬件要求因运行 Dashy 的位置和方式而异。</font><font style="vertical-align: inherit;">一般来说，在裸机系统或 Docker 容器上，1GB 内存应该绰绰有余，而根据您是否使用自己的资产，1GB 磁盘空间也应该足够。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如果您使用其中一种一键式云部署方法，通过 CDN 或使用静态托管提供商提供应用程序，则没有具体要求，因为构建的应用程序只是一系列静态 JS 文件，因此重量非常轻。</font></font></p>
